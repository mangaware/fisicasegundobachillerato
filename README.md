# App local de Fisica de 2o de Bachillerato

Base de trabajo para una app local enfocada en teoria, formulas, ejercicios y simulacros de Fisica de 2o de Bachillerato.

## Estructura inicial

- `src/`: interfaz base de la app.
- `content/apuntes/`: PDFs, fotos o texto de teoria.
- `content/ejercicios/`: ejercicios resueltos o propuestos.
- `content/examenes/`: controles y modelos EVAU.
- `content/formulas/`: formularios y resumentes finales.

## Proximo uso

1. Anade tus materiales en las carpetas de `content/`.
2. Instala Node.js y npm si no estan disponibles en tu equipo.
3. Ejecuta `npm install`.
4. Ejecuta `npm run dev`.

## MVP sugerido

1. Pantalla de inicio con progreso.
2. Temario por bloques.
3. Formularios por tema.
4. Ejercicios resueltos y practica.
5. Simulacro tipo examen.
