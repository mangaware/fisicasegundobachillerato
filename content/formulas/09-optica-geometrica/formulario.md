# 09 Óptica geométrica

## Formulario rápido

- Ecuación de las lentes delgadas :: \frac{1}{s'}-\frac{1}{s}=\frac{1}{f'}
- Ecuación del fabricante de lentes :: \frac{1}{f'}=(n-1)\left(\frac{1}{R_1}-\frac{1}{R_2}\right)
- Aumento lateral :: M=\frac{y'}{y}=\frac{s'}{s}
- Potencia de una lente :: C=\frac{1}{f'}
- Signo de lente convergente :: f'>0
- Signo de lente divergente :: f'<0
