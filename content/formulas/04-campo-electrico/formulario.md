# 04 Campo eléctrico

## Formulario rápido

- Ley de Coulomb :: F=k\frac{Q_1Q_2}{r^2}
- Campo eléctrico de una carga puntual :: E=k\frac{Q}{r^2}
- Fuerza sobre una carga en un campo :: F=qE
- Potencial eléctrico de una carga puntual :: V=k\frac{Q}{r}
- Energía potencial eléctrica :: E_p=qV
- Trabajo interno del campo :: W_{\mathrm{int}}(A\to B)=q\,[V(A)-V(B)]
- Flujo eléctrico :: \Phi=\int \vec{E}\cdot d\vec{S}
- Ley de Gauss :: \Phi=\frac{Q_{\mathrm{encerrada}}}{\varepsilon_0}
- Relación campo-potencial en campo uniforme :: |E|=\frac{\Delta V}{\Delta x}
- Aceleración de una partícula cargada :: a=\frac{|q|\,|E|}{m}
- Energía cinética por diferencia de potencial :: |q\,\Delta V|=\frac{1}{2}mv_f^2
