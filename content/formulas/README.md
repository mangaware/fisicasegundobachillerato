# Formularios por tema

Cada carpeta de `content/formulas` incluye un `formulario.md` con las expresiones clave del tema.

- El objetivo es tener un repaso rápido separado de la teoría larga.
- Estas fichas salen de las secciones `Formulas clave` ya curadas en `content/temas`.
- Si añades nuevas ecuaciones o constantes, actualiza también la ficha del tema correspondiente para no desincronizar teoría y formulario.
- El formato recomendado es `Etiqueta :: codigo TeX` para que la app lo renderice con lenguaje matematico real.
