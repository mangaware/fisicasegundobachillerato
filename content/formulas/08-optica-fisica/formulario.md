# 08 Óptica física

## Formulario rápido

- Índice de refracción :: n=\frac{c}{v}
- Relación entre índice y longitud de onda :: n=\frac{\lambda_0}{\lambda}
- Ley de Snell :: n_1\sin i=n_2\sin r
- Forma equivalente con velocidades :: \frac{\sin i}{\sin r}=\frac{v_1}{v_2}=\frac{n_2}{n_1}
- Angulo limite :: L=\arcsin\!\left(\frac{n_2}{n_1}\right),\quad n_1>n_2
- Desviacion angular en prisma :: \delta=i_1+r_2-\varphi
- Angulo del prisma :: \varphi=i_2+r_1
