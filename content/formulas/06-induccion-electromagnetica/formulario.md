# 06 Inducción electromagnética

## Formulario rápido

- Flujo magnético :: \Phi=BS\cos\theta
- Fuerza electromotriz media :: \varepsilon_{\mathrm{media}}=-\frac{\Delta\Phi}{\Delta t}
- Fuerza electromotriz instantanea :: \varepsilon=-\frac{d\Phi}{dt}
- Fem inducida en una barra en MRU :: \varepsilon=BLv
- Relación entre amplitudes de onda electromagnética :: c=\frac{E_0}{B_0}
- Velocidad de la luz en el vacío :: c=\frac{1}{\sqrt{\varepsilon_0\mu_0}}
- Relación onda electromagnética :: \lambda=cT=\frac{c}{f}
- Campo eléctrico de una OEM :: E(x,t)=E_0\sin(\omega t-kx)
- Campo magnético de una OEM :: B(x,t)=B_0\sin(\omega t-kx)
