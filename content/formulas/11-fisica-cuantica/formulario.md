# 11 Física cuántica

## Formulario rápido

- Energía cuantizada :: E=hf
- Trabajo de extraccion :: W_e=hf_0
- Efecto fotoeléctrico :: E_{\gamma}=W_e+E_c
- Energía cinética máxima del fotoelectron :: E_c=eV_0=h(f-f_0)
- Potencial de frenado :: V_0=\frac{h}{e}f-\frac{h}{e}f_0
- Relación de De Broglie :: p=\frac{h}{\lambda}
- Longitud de onda de De Broglie :: \lambda=\frac{h}{p}
- Principio de indeterminacion :: \Delta x\,\Delta p\geq\frac{h}{4\pi}
- Formula de Rydberg para Bohr :: \frac{1}{\lambda}=R_H\left(\frac{1}{n^2}-\frac{1}{m^2}\right),\quad m>n
