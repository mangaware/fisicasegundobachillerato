# 05 Campo magnético

## Formulario rápido

- Fuerza magnética :: \vec{F}_m=q\,(\vec{v}\times\vec{B})
- Modulo de la fuerza magnética :: |F_m|=|q|\,|v|\,|B|\sin\alpha
- Radio de la trayectoria circular :: R=\frac{mv}{qB}
- Período del movimiento circular :: T=\frac{2\pi m}{qB}
- Frecuencia ciclotron :: f=\frac{qB}{2\pi m}
- Fuerza de Lorentz :: \vec{F}_L=q\,(\vec{E}+\vec{v}\times\vec{B})
- Selector de velocidades :: v=\frac{|E|}{|B|}
- Campo de un hilo rectilineo largo :: B=\frac{\mu_0 I}{2\pi r}
- Fuerza entre corrientes paralelas :: \frac{|F|}{L}=\frac{\mu_0 I_1 I_2}{2\pi d}
- Definicion operativa de tesla :: 1\ \mathrm{T}=\frac{1\ \mathrm{N}}{1\ \mathrm{A}\cdot\mathrm{m}}
