# 10 Relatividad especial

## Formulario rápido

- Factor de Lorentz :: \gamma=\frac{1}{\sqrt{1-\frac{v^2}{c^2}}}
- Transformacion espacial de Lorentz :: x'=\gamma(x-vt)
- Transformacion temporal de Lorentz :: t'=\gamma\left(t-\frac{vx}{c^2}\right)
- Dilatacion temporal :: \Delta t=\gamma\,\Delta t'
- Contraccion longitudinal :: L'=\gamma L
- Longitud medida en movimiento :: L=\frac{L'}{\gamma}
- Masa relativista :: m=\gamma m_0
- Cantidad de movimiento relativista :: p=\gamma m_0 v
- Energía en reposo :: E_0=m_0c^2
- Energía cinética relativista :: E_c=(\gamma-1)E_0
- Energía total relativista :: E_T=mc^2=E_0+E_c
