# 12 Física nuclear

## Formulario rápido

- Relación entre semivida y constante de desintegración :: T_{1/2}=\frac{\ln 2}{\lambda}
- Vida media :: \tau=\frac{1}{\lambda}
- Nucleos no desintegrados :: N(t)=N_0e^{-\lambda t}
- Forma con semivida :: N(t)=N_0\,2^{-t/T_{1/2}}
- Actividad de una muestra :: A(t)=\lambda N(t)
- Actividad con semivida :: A(t)=A_0\,2^{-t/T_{1/2}}
- Masa remanente :: m(t)=m_0\,2^{-t/T_{1/2}}
- Número masico :: A=Z+N
