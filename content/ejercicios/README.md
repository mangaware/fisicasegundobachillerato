# Banco de ejercicios

Los ejercicios de esta carpeta son una version curada y resumida de enunciados reales de PAU/EBAU.

- Cada tema usa `pau-ebau.md` como banco base para el modo práctica.
- El objetivo no es copiar exámenes completos, sino separar ejercicios reutilizables por tema.
- Si añades ejercicios nuevos, conserva siempre la fuente y la pista de arranque.
