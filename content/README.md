# Contenido para la app

Coloca aqui el material del curso para convertirlo despues en contenido navegable dentro de la aplicacion.

- `00-inbox/`: carpeta rapida para soltar archivos sin ordenar.
- `apuntes/`: teoria por tema.
- `ejercicios/`: problemas resueltos o para practicar.
- `examenes/`: examenes de clase o EVAU.
- `formulas/`: formularios, tablas y constantes.

## Flujo recomendado

1. Si tienes todo mezclado, mete primero los archivos en `00-inbox/`.
2. Cuando puedas, mueve cada archivo a su carpeta por tipo y tema.
3. Usa nombres claros, por ejemplo `tema-03-resumen-gravitacion.pdf` o `examen-evau-2024-modelo-a.pdf`.
